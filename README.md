# 🧮 Calculadora Web

Projeto de uma calculadora desenvolvido durante meus estudos em desenvolvimento web, com foco em praticar JavaScript na manipulação do DOM, eventos e lógica de programação.

A ideia foi simular uma aplicação real simples, trabalhando tanto a parte visual quanto o comportamento funcional da interface.

---

## 🚀 Funcionalidades

- Operações matemáticas básicas:
  - Adição (+)
  - Subtração (-)
  - Multiplicação (*)
  - Divisão (/)
- Botão de limpar tela (AC)
- Inversão de sinal (+/-)
- Suporte ao teclado (tecla Enter para calcular)
- Interface inspirada em calculadoras mobile

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Grid Layout)
- JavaScript (Vanilla JS)
